Links

There are two different LESS files that can be included in your SPF site, spf.less (spf.min.css) or spf-with-icons.less (spf-with-icons.min.css). The latter includes icons that are automatically added to your page using pseudo-elements, attribute selectors and a spritesheet. Here are some examples: a .pdf file, a .png file, a .xls file.

Responsive elements

.responsive-scroll-region

This class can be placed on a wrapping element to any content and it will become a horizontally scrollable section when sized down to the mobile breakpoint.

.responsive-table-two-column

This class will transform a table when the mobile breakpoint is hit into a 2 column table with the headers for each element on the left. Note that the header can wrap to the next line and it won't increase the size of the row.

Customisations

The following classes can be used to modify the size of the header column. This will help to avoid wrapping where possible.

  • .header-30
  • .header-40
  • .header-60
  • .header-70
